# Codebook

## Dataset
Pesquisadores em sistemas neurosimbólicos (sistemas fuzzy) na América Latina.

## Unidade de análise
Pesquisador, projeto de pesquisa, universidade, ano.

## Variáveis

nome
Tipo: string
Descrição: nome do pesquisador

universidade
Tipo: string
Descrição: instituição de vínculo

titulo_pesquisa
Tipo: string
Descrição: título da tese ou projeto

tecnica
Tipo: lista
Descrição: método utilizado na pesquisa
